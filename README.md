# Firegram

Firegram es una plataforma para subir imagenes realizada con React y Firebase.


## Authors

- [@tomassanchezz](https://www.github.com/tomassanchezz)

  
## Features

- Subida de imagenes


  
## Installation 

Instalar el proyecto con npm install

```bash 
  npm install 
```
    
## Tech Stack

**Client:** React

**Server:** Node

**Animations:** Framer
